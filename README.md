# 🎮 Rock Paper Scissors Game (Python)

An interactive command-line game where the user plays against the computer using random choices of rock, paper, or scissors.

---

----- Features
- Play against the computer
- Random computer selection using Python’s `random` module
- Help option explaining game rules
- Input validation to handle incorrect entries
- Emoji-based output for a better user experience
- Clear win/lose/tie logic

----- Game Rules
- Rock beats Scissors  
- Scissors beat Paper  
- Paper beats Rock
  
-----  How It Works
1. The user selects rock, paper, or scissors  
2. The computer randomly selects its choice  
3. The program compares both choices  
4. The winner is displayed based on game rules  

----- Technologies Used
- Python
- Random module
- Conditional statements (if / elif / else)
- User input handling
